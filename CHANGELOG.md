## 1.0.0

- First stable release of `rgb_form_kit`.
- Added reusable form components built on `flutter_form_builder`.
- Added `Formable<T>` for custom field composition.
- Added `saveAndValidateWithAutoFocusOnError()` extension to improve
  validation UX by scrolling/focusing first invalid field.
- Added package documentation: README and usage guide.
